# 🧠 AI-Powered Health Advisor System

## 🚀 Overview
This is a full-stack AI-based web application that provides health-related suggestions and BMI analysis using an integrated AI chatbot.

---

## 🔥 Features
- BMI Calculation
- AI Chatbot for health advice
- User Authentication (Login/Signup)
- Chat History stored in database
- User-specific chat system
- View, load, and delete previous chats

---

## 🛠 Tech Stack
- Backend: Spring Boot
- Frontend: Thymeleaf + Bootstrap
- Database: MySQL
- AI Integration: Gemini API

---

## ⚙️ How to Run
1. Clone the repository
2. Open in STS / IntelliJ
3. Configure MySQL in application.properties
4. Run the project
